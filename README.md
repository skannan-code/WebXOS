# *WebXOS - The Project of a Lifetime*
WebXOS is an open-source webOS project afar from the galaxies and cosmos explorable. In this versatile platform, you can explore half a dozen apps that allow you to do amazing things, while not having to worry about additional fees or threats to your security. So don't wait, and be the first to witness this amazing operating system in full screen!

# What should I expect?
When exploring this magnificent operating system, you have a variety of choices to explore. To be exact, you can use the Notepad app to jot down ideas and thoughts, the Google app to search for anything if you like to research, the Calculator app if you wish to calculate complex equations, the IP Helpdesk app if you need to know your exact IP address, the Settings app that allows you to change the brightness, contrast, and wallpaper of your screen if you would like to personalize it, and the Camera app if you want to take a quick selfie that is automatically downloaded into your local storage.

# Is there anything else I should know?
In fact, yes. When exploring WebXOS, please use modern browsers such as Google Chrome and Microsoft Edge. This project was tested exculsively on these two web browsers, so I cannot guarantee it will work as well on an older browser.
